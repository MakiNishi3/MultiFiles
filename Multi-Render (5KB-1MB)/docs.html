Introduction

OrdinalNum.js is a JavaScript library for computing with transfinite ordinal numbers beyond ordinary arithmetic. It supports the full Cantor Normal Form hierarchy, Veblen functions, Buchholz's ψ function, and the Small Veblen Ordinal (SVO).

All ordinals are represented as instances of OrdinalNum. The library uses decimal.js internally for finite precision arithmetic (precision 40, rounding mode 4).

The library exposes its API through the global OrdinalLib object in browser environments.

Setup

// Browser — include decimal.js first, then OrdinalNum.js
<script src="https://cdnjs.cloudflare.com/ajax/libs/decimal.js/10.4.3/decimal.min.js"></script>
<script src="https://raw.githubusercontent.com/MakiNishi3/OrdinalNum.js/refs/heads/gh-pages/OrdinalNum.js"></script>

// Then access via:
const { OrdinalNum, omega, epsilon } = OrdinalLib;

// Node.js
const OrdinalLib = require('./OrdinalNum.js');

Constructors

Example

omega()

OrdinalLib.omega() → OrdinalNum

Returns the first infinite ordinal ω (omega).

epsilon(subscript?)

OrdinalLib.epsilon(subscript?) → OrdinalNum

Returns ε_subscript — the subscript-th fixed point of α ↦ ω^α. Default subscript is 0.

zeta(subscript?)

OrdinalLib.zeta(subscript?) → OrdinalNum

Returns ζ_subscript — the subscript-th fixed point of α ↦ ε_α.

gamma(subscript?)

OrdinalLib.gamma(subscript?) → OrdinalNum

Returns Γ_subscript (Feferman–Schütte ordinal Γ₀ for subscript=0).

ph(alpha, beta)

OrdinalLib.ph(alpha, beta) → OrdinalNum

Two-argument Veblen function φ(α, β). When α=0 returns ω^β.

psi(nu, alpha) / psi(alpha)

OrdinalLib.psi(nu?, alpha) → OrdinalNum

Buchholz's ψ collapsing function ψ_ν(α). When nu is omitted, defaults to 0.

svoOrdinal()

OrdinalLib.svoOrdinal() → OrdinalNum

Returns the Small Veblen Ordinal — the limit of the Veblen hierarchy.

Arithmetic

Ordinal arithmetic is NOT commutative. Addition and multiplication follow ordinal rules, not natural number rules.

.add(b)

a.add(b) → OrdinalNum

Ordinal addition. 1 + ω = ω, but ω + 1 ≠ ω.

.mul(b)

a.mul(b) → OrdinalNum

Ordinal multiplication. 2 × ω = ω, but ω × 2 = ω·2.

.pow(b)

a.pow(b) → OrdinalNum

Ordinal exponentiation.

Ordinal Hierarchy

From smallest to largest, the ordinals representable by this library:

0, 1, 2, ... n                    ← finite ordinals
ω, ω+1, ω·2, ω^2, ω^ω, ...       ← CNF ordinals
ε₀ = φ(1,0)                       ← first epsilon number
ε₁, ε₂, ... εω, ...              ← epsilon hierarchy
ζ₀ = φ(2,0)                       ← first zeta number
Γ₀ = φ(ω^ω, 0)                    ← Feferman–Schütte ordinal
φ(Γ₀+1, 0), ...                   ← beyond Γ₀
SVO                                ← Small Veblen Ordinal (top)
